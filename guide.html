Welcome to ai-enclave

ai-enclave is a secure, isolated AI development environment running in Docker. It provides Claude Code CLI, code-server (browser-based VS Code), GitHub CLI, and Bitwarden CLI — all inside a sandboxed Ubuntu 24.04 container.

Your workspace is at /workspace inside the container. API keys and secrets are managed via Bitwarden CLI and never stored in plain text.

Open code-server →
2
First Time Setup 初回のみ

Claude Code 認証

コンテナ内で claude を実行するとブラウザ認証が始まります。

claude

表示されるURLをブラウザで開き、Anthropicアカウントでログインしてください。

プロジェクトのclone

git clone <repo-url> /workspace/<project-name>
cd /workspace/<project-name>
GitHub認証には gh auth login を使用してください。
3
API Keys & Secrets

シークレットは Bitwarden CLI で管理します。平文ファイルへの保存は避けてください。

Bitwardenログイン & アンロック

bw login
export BW_SESSION=$(bw unlock --raw)

APIキーを環境変数に設定

# 例: xAI APIキー
export XAI_API_KEY=$(bw get password "xAI API Key")

# 例: Anthropic APIキー
export ANTHROPIC_API_KEY=$(bw get password "Anthropic API Key")
環境変数はコンテナ再起動で消えます。~/.bashrc に bw get コマンドを追記しておくと便利です。
4
Using code-server

ブラウザで http://localhost:8080 にアクセスしてください。

パスワード確認（必要な場合）

cat ~/.config/code-server/config.yaml

password: の行に記載されたパスワードを使用してください。

start.batで起動した場合、--auth none オプションでパスワード不要になっています。

ターミナルから Claude Code を使う

cd /workspace/<project>
claude
5
For Power Users

コンテナに直接接続

docker exec -it ai-enclave bash

tmux で複数セッション管理

# 新規セッション
tmux new -s main

# セッション一覧
tmux ls

# アタッチ
tmux attach -t main

code-serverログ確認

tail -f /tmp/code-server.log
6
Stopping

stop.bat をダブルクリック — Windowsから簡単停止

コマンドラインから

docker compose stop
コンテナを完全に削除する場合: docker compose down
再起動は start.bat をダブルクリックするだけです。
